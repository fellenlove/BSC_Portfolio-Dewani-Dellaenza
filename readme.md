### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- Git installed on your system

### Quick Start

1. **Clone the Repository**

   ```bash
   git clone https://github.com/fellenlove/LogicLens.git
   cd LogicLens
   ```

2. **Open in Browser**
   - Double-click `index.html`, or
   - Use a local server (recommended for best performance):
     ```bash
     python -m http.server 8000
     ```
     Then navigate to `http://localhost:8000` in your browser.

3. **Explore**
   - Toggle the theme using the sun/moon icon in the header
   - Test the responsive design by resizing your browser or opening on mobile
   - Check out the project gallery and explore each featured project
